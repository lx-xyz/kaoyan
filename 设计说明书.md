# 考研一体化学习辅助系统 设计说明书

## 摘 要

本系统是一款面向考研学生的网页版一体化学习辅助工具，基于 Python Flask 框架和 Bootstrap 前端技术栈开发，部署于 PythonAnywhere 云平台。系统集成了题目管理、做题练习、错题追踪、单词记忆、笔记记录、专注计时、批量导入等功能模块，并引入艾宾浩斯记忆曲线算法实现智能化复习计划。系统支持多用户注册登录与游客体验模式，提供管理后台用于全局配置，适用于不同考研方向的考生自定义使用。

**关键词**：考研；学习辅助；Flask；记忆曲线；Web 应用

---

## 一、项目背景与目标

### 1.1 项目背景

随着考研竞争日益激烈，考生需要高效管理大量的真题、错题和单词学习内容。传统的纸质笔记本和分散的手机 App 难以满足一体化学习的需求。本系统旨在为考生提供一个集题目管理、智能复习、学习计时于一体的在线学习平台。

### 1.2 目标用户

- 考研备考学生（以 22408 计算机专硕为例，支持自定义考试科目）
- 需要系统化管理学习内容的自学者

### 1.3 核心目标

1. 提供真题录入、做题、自动判分的完整流程
2. 基于艾宾浩斯遗忘曲线的错题追踪与单词记忆
3. 番茄钟计时器记录每日学习时长
4. 多用户支持与管理后台

## 五、核心算法设计

### 5.1 艾宾浩斯记忆曲线

本系统在错题追踪和单词记忆模块中采用了基于艾宾浩斯遗忘曲线的间隔复习算法。

**算法原理**：根据人类记忆遗忘规律，在即将遗忘的时间点进行复习，以最小复习次数达到最佳记忆效果。

**间隔设计**：

| 阶段 | 间隔 | 说明 |
|------|------|------|
| 0 | 10分钟 | 首次错误/不认识 |
| 1 | 1天 | 第一次正确复习后 |
| 2 | 3天 | 第二次正确复习后 |
| 3 | 7天 | 第三次正确复习后 |
| 4 | 15天 | 第四次正确复习后 |
| 5 | 30天 | 第五次正确复习后 → 标记"已掌握" |

**核心代码逻辑**（`services/spaced_repetition.py`）：

```python
STAGE_INTERVALS = {0: 1, 1: 3, 2: 7, 3: 15, 4: 30}

def advance_stage(current_stage, result):
    if result == "correct":
        new_stage = min(current_stage + 1, 5)  # 推进
    elif result == "wrong":
        new_stage = 0  # 重置
    else:
        new_stage = current_stage  # 模糊，保持
    next_date = now + timedelta(days=STAGE_INTERVALS.get(new_stage, 1))
    return new_stage, next_date
```

## 七、功能模块详细设计

### 7.1 用户系统

- **注册**：用户名（2~64字符）+ 密码（≥6位），系统自动生成邮箱
- **登录**：区分普通用户/管理员登录模式
- **游客模式**：自动生成临时账号，退出后所有数据自动清除
- **管理员**：`is_admin=True`，可访问管理后台
- **个人中心**：编辑资料、设置考试日期、自定义励志语录

### 7.2 题目管理

- 录入表单支持科目选择、题型选择（可自定义输入）、LaTeX公式、截图粘贴
- 题库列表支持按科目、年份、题型筛选，分页显示
- 题目详情页展示完整信息，正确答案高亮标注

### 7.3 做题功能

- **随机出题模式**：选择科目后随机抽取题目
- **选题练习模式**：浏览所有题目，手动选择要做的题
- 选择题：点击选项卡片作答
- 填空题/解答题：输入答案文本
- 提交后：正确选项变绿+标签，错误选项变红+标签，解析显示在下方

### 7.4 错题追踪

- 做错的题自动写入`mistake_item`表
- 错题本列表显示科目、题型、错误次数、掌握度
- 复习页面按记忆曲线出题（到期的错题优先）
- 答对推进阶段，答错重置

### 7.5 单词记忆

- **添加**：批量输入（格式：单词|释义|音标）
- **学习新词**：卡片式显示，认识→加入复习周期，不认识→回炉重学
- **复习**：到期单词卡片式自评（认识/不认识），不认识本轮结束重新出现
- **批量操作**：全选/取消、学习所选、切换状态、重置、删除

### 7.6 笔记记录

- Quill.js富文本编辑器，所见即所得
- 支持加粗、斜体、下划线、字体颜色、背景色、列表、链接、图片
- Ctrl+V图片自动上传
- 关联科目、分类标签、置顶

### 7.7 专注计时

- 圆形进度环倒计时显示
- 自定义小时+分钟时长
- 可选关联科目
- 计时结束自动记录时长
- 今日学习分钟数和番茄数实时统计

### 7.8 批量导入

- 粘贴文本：自动识别题号、ABCD选项、答案、解析
- Excel模板下载：标准列格式，上传后逐行解析入库
- 导入历史记录

### 7.9 学习统计

- 总学习时长、累计天数、总做题数、已掌握单词
- 近7天学习时长柱状图（Chart.js）
- 各科目学习时间环形图
