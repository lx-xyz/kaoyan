# 考研一体化学习辅助系统 设计说明书

## 摘 要

本系统是一款面向考研学生的网页版一体化学习辅助工具，基于 Python Flask 框架和 Bootstrap 前端技术栈开发，部署于 PythonAnywhere 云平台。系统集成了题目管理、做题练习、错题追踪、单词记忆、笔记记录、专注计时、批量导入等功能模块，并引入艾宾浩斯记忆曲线算法实现智能化复习计划。系统支持多用户注册登录与游客体验模式，提供管理后台用于全局配置，适用于不同考研方向的考生自定义使用。

**关键词**：考研；学习辅助；Flask；记忆曲线；Web 应用

---

## 一、项目背景与目标

### 1.1 项目背景

随着考研竞争日益激烈，考生需要高效管理大量的真题、错题和单词学习内容。传统的纸质笔记本和分散的手机 App 难以满足一体化学习的需求。本系统旨在为考生提供一个集题目管理、智能复习、学习计时于一体的在线学习平台。

### 1.2 目标用户

- 考研备考学生（支持自定义考试科目，默认含数学、英语、政治三科）
- 需要系统化管理学习内容的自学者

### 1.3 核心目标

1. 提供真题录入、做题、自动判分的完整流程
2. 基于艾宾浩斯遗忘曲线的错题追踪与单词记忆
3. 番茄钟计时器记录每日学习时长
4. 多用户支持与管理后台

## 五、核心算法设计

### 5.1 艾宾浩斯记忆曲线

本系统在错题追踪和单词记忆两个模块中采用了基于艾宾浩斯遗忘曲线的间隔复习算法。

**算法原理**：根据人类记忆遗忘规律，在即将遗忘的时间点进行复习，以最小复习次数达到最佳记忆效果。每次正确复习后延长间隔，错误则重置。

```
              认识/正确                              认识/正确
  ┌──────┐    ────►    ┌──────┐    ────►    ┌──────┐
  │阶段0 │            │阶段1 │            │阶段2 │
  │10min │            │ 1天  │            │ 3天  │
  └──┬───┘    ◄────    └──────┘    ◄────    └──┬───┘
     │        不认识      │        不认识      │
     └────────────────────┴────────────────────┘
                   (重置为阶段0)

    继续推进：阶段3(7天) → 阶段4(15天) → 阶段5(已掌握 ✓)
```

**间隔设计表**：

| 阶段 | 间隔 | 含义 |
|------|------|------|
| 0 | 10分钟 | 首次错误/不认识 |
| 1 | 1天 | 第一次正确复习 |
| 2 | 3天 | 第二次正确复习 |
| 3 | 7天 | 第三次正确复习 |
| 4 | 15天 | 第四次正确复习 |
| 5 | 已掌握 | 第五次正确后标记掌握 |

**核心代码逻辑**（`services/spaced_repetition.py`）：

```python
STAGE_INTERVALS = {0: 10/1440, 1: 1, 2: 3, 3: 7, 4: 15}

def advance_stage(current_stage, result):
    """推进记忆阶段"""
    if result == "correct":
        new_stage = min(current_stage + 1, 5)    # 推进
    elif result == "wrong":
        new_stage = 0                             # 重置
    else:
        new_stage = current_stage                 # 保持
    interval_days = STAGE_INTERVALS.get(new_stage, 1)
    next_date = datetime.utcnow() + timedelta(days=interval_days)
    return new_stage, next_date
```

### 5.2 单词学习回炉算法

```
┌──────────────┐    认识    ┌──────────────┐
│  新词池       │ ────────► │  复习周期      │
│  (stage=-1)  │           │  (stage=0~5)  │
└──────┬───────┘           └──────────────┘
       │ 不认识                 ▲
       │ (本轮回炉)             │
       └────────────────────────┘
       (本轮全部过完后重新出现)
       
学习会话流程：
  第1轮: 词A 词B 词C 词D 词E
         ↓     ↓     ↓    ↓    ↓
       认识  不认识  认识 不认识 认识
       
  第2轮(回炉): 词B 词D  →  全部认识 → 完成!
```

## 七、功能模块详细设计

### 7.1 用户系统

- **注册流程**：输入用户名(2~64字符)+密码(≥6位)→验证唯一性→bcrypt加密存储→自动生成邮箱→自动登录→跳转仪表盘
- **登录流程**：区分普通用户/管理员模式；管理员账号在普通模式下会提示切换；验证通过→跳转仪表盘
- **游客模式**：一键生成临时账号→所有功能可用→退出后数据自动清除(包括做题记录、单词、科目等)
- **权限控制**：`@login_required` 保护路由，`is_admin` 控制管理后台可见性

### 7.2 题目管理

- 录入表单：科目下拉(根据选择自动切换题型选项)+年份/来源(可选)+题型(输入框+datalist，支持自定义)+题号(可选)+难度(可选，默认不设置)+题干(textarea，支持KaTeX公式和截图粘贴)+ABCD选项(可选)+答案+标签+解析
- 题库列表：卡片式展示，科目/年份/题型筛选，分页浏览
- 详情页：完整题目信息展示，正确答案高亮

### 7.3 做题功能

```
用户进入做题页
    │
    ├── 随机出题模式──────────────────────────────┐
    │   选科目 → 随机出一题 → 作答 → 判分 → 下一题  │
    │                                            │
    ├── 选题练习模式──────────────────────────────┤
    │   浏览题目列表 → 点选题目 → 作答 → 判分        │
    │                                            │
    判分逻辑：                                    │
    ├── 选择题：比较选项字母(A/B/C/D)               │
    ├── 填空题：比较答案文本                         │
    ├── 正确：选项变绿+边框+标签 → 显示解析          │
    └── 错误：所选变红+正确答案变绿 → 自动入错题本    │
```

### 7.4 错题追踪

```
做错一题
    │
    ▼
检查错题本是否已有该题
    │
    ├── 已有 → wrong_count+1, mastery_level-1, 重置复习时间
    │
    └── 没有 → 新建错题记录, mastery_level=0, 10分钟后复习
    │
    ▼
错题本列表（按科目/状态筛选）
    │
    ├── 待复习(红色) ──→ 进入复习页
    │
    ├── 阶段X(灰色) ────→ 等待到期
    │
    └── 已掌握(绿色) ──→ 可手动移回待复习
    │
    ▼
复习页：题目 → 作答 → 正确推进阶段 / 错误重置 → 下一题
```

### 7.5 单词记忆

```
添加单词(stage=-1)
    │
    ▼
学习新词（卡片式UI）
    │
    ├── 认识 → stage=0, 1天后复习
    │
    └── 不认识 → 保持stage=-1, 加入回炉队列
    │
    ▼
本轮全部过完
    │
    ├── 有不认识的 → 回炉词重新出现
    │
    └── 全部认识 → 学习完成弹窗
    │
    ▼
复习（到期词卡片复习）
    │
    ├── 认识 → 推进记忆阶段(0→1→2→3→4→5)
    │
    └── 不认识 → 重置为阶段0
```
