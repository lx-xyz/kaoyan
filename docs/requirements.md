# 功能需求规格

## 项目概述
为 22408（计算机专硕）考研学生打造的网页版一体化学习辅助系统。

**科目范围**：英语二、数学二、408计算机综合  
**用户模式**：多用户（注册/登录）  
**部署目标**：免费云平台（PythonAnywhere）

---

## 模块需求

### 1. 用户系统
- 用户注册（用户名、邮箱、密码）
- 用户登录 / 注销
- 个人中心：昵称、目标院校、目标专业、每日学习目标
- 密码加密存储（bcrypt）

### 2. 真题管理
- 按科目/年份浏览题目
- 手动录入题目（表单填写）
- 编辑/删除已有题目
- 题干支持 Markdown 和 KaTeX 数学公式
- 题干支持粘贴图片

### 3. 做题功能
- 逐题练习模式
- 选择题自动判断对错
- 显示正确答案和解析
- 记录做题历史
- 做错自动加入错题本

### 4. 错题追踪
- 自动收集错题
- 按科目/掌握度筛选
- 艾宾浩斯记忆曲线复习计划
- 错题备注
- 连续答对自动标记"已掌握"

### 5. 单词记忆
- 单词本（添加/编辑/删除）
- 卡片式复习（正面英文，翻转看释义）
- 自评：认识/模糊/不认识
- 记忆曲线推进
- 批量添加单词

### 6. 笔记记录
- Markdown 编辑器
- 笔记分类标签
- 关联科目
- 置顶功能

### 7. 专注计时
- 番茄钟：25分钟专注 + 5分钟休息
- 每4个番茄钟长休息15分钟
- 记录学习时段
- 关联学习科目

### 8. 批量导入
- Excel 模板下载（真题模板 + 单词模板）
- 上传 Excel 批量导入
- 数据校验 + 错误反馈
- 导入历史记录

### 9. 习题册
- 自建习题册
- 从题库选题组卷
- 习题册内排序

### 10. 学习仪表盘
- 今日学习时长
- 待复习错题数/单词数
- 今日做题数
- 快速入口按钮

### 11. 学习统计
- 近7天/30天学习时长柱状图
- 各科目学习时间饼图
- 累计数据汇总
