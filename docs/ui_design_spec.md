# UI 设计规范

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 Primary | `#FFA000` | 导航栏、主要按钮、强调元素 |
| 主色浅 Light | `#FFC107` | 悬停态、次要强调 |
| 主色背景 | `#FFF8E1` | 卡片底色、提示框背景 |
| 主色边框 | `#FFE082` | 边框、分割线 |
| 成功色 | `#4CAF50` | 答对提示、已掌握标记 |
| 危险色 | `#F44336` | 答错提示、待复习标记 |
| 信息色 | `#2196F3` | 链接、单词卡片 |
| 页面背景 | `#FAFAFA` | 全局背景 |
| 正文文字 | `#333333` | 标题、正文 |
| 次要文字 | `#757575` | 描述、辅助信息 |

## 字体

- 系统默认字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "Microsoft YaHei", sans-serif`
- 中文优先使用微软雅黑
- 标题加粗 `font-weight: bold` / `fw-bold`
- 正文 `font-size: 1rem` (16px)

## 间距

- 卡片内边距：`1.5rem` (p-4)
- 页面内容区：`mt-4 mb-5`
- 卡片间距：`g-4` (1.5rem gap)
- 按钮间距：`gap-3` (1rem)

## 组件规范

### 导航栏
- 背景色：`#FFA000`
- 文字：白色
- 高度：标准 Bootstrap navbar
- 底部阴影：`0 2px 8px rgba(255, 160, 0, 0.2)`
- 品牌名左侧有书本图标

### 卡片
- 圆角：`12px`
- 背景：白色
- 阴影：`shadow-sm`
- 悬停效果：上移 2px + 阴影加深
- 左侧色条：4px 宽，用于统计卡片

### 按钮
- 主按钮：黄色背景 `#FFA000` + 白色文字
- 悬停：`#FF8F00`
- 轮廓按钮：黄色边框 + 黄色文字
- 大按钮：`btn-lg` + `px-4`

### 统计数字
- 大号数字：`2.5rem` 字重 `bold`
- 颜色与卡片左侧色条一致

### Flash 消息
- Bootstrap alert 组件
- 成功：绿色，危险：红色，信息：蓝色，警告：黄色
- 可关闭（带 × 按钮）

## 响应式断点

- 移动端 `< 768px`：统计数字缩小，导航折叠
- 平板 `768px - 992px`：卡片2列
- 桌面 `> 992px`：卡片4列

## 动画

- 卡片入场：`fadeInUp` 0.4s（从下方淡入上移）
- 番茄钟圆环：CSS 动画
